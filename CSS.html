CSS Specificity คือค่าความสำคัญที่จะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ
เมื่อมีการกำหนด CSS ที่ element เดียวกัน โดยจะเลือก Style ที่มีค่าความสำคัญมากที่สุด (มีการกำหนดเจาะจงที่สุด)
ถ้าเจอที่การกำหนด style ที่ element ที่ซ้ำกัน หรือมีค่าความสำคัญเท่ากัน จะเลือก selector ที่อยู่ข้างล่าง
จัดกลุ่มการเขียน Style ได้ 4 กลุ่ม

ค่าความสำคัญ เป็นดังนี้

Element selector
Class selector
ID selector
Inline
Element selector (ค่าความสำคัญน้อยที่สุด) Class selector ID selectors Inline style (ค่าความสำคัญมากที่สุด)
แบ่งคะแนนเค่าความสำคัญเป็นแบบนี้ x,y,z,t (เลข 4 ตัว Style 4 กลุ่ม )
x = Inline style
y = ID selector
z = Class selector
t = Element selector
การคิดคะแนน จะดูจากฝั่งซ้าย เป็นค่าสูงสุด และลดลงตามลำดับ

Example

HTML file

กำหนด Style แบบ Inline style ให้ตัวอักษรมีสีแดง

1,0,0,0

CSS file

กำหนด Style แบบ Element selector ให้ตัวอักษรมีสีเหลือง

0,0,0,1

Output

Inline style มีค่าความสำคัญมากสุด ผลลัพธ์ออกมา ตัวอักษรจะมีสีแดง

HTML file

div มี id = test

CSS file

กำหนด style

div#test h1 : 0,1,0,2
div h1 : 0,0,0,2
h1 : 0,0,0,1

Output

โดยจะเห็นว่า div#test h1 มีการกำหนดแบบเจาะจงที่สุด และมีค่าความสำคัญมากที่สุด ตัวอักษรจึงออกมาเป็นสีส้ม
