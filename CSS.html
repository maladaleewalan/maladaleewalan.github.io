CSS Specificity คือค่าความสำคัญที่จะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ
โดยจะเลือก Style ที่มีค่าความสำคัญมากที่สุด ถ้ามีค่าเท่ากัน จะเลือกอันที่อยู่ข้างล่าง
จัดกลุ่มการเขียน Style ได้ 4 กลุ่ม Inline style ID selectors Class selector , Pseudo-classes Element selector
ค่าความสำคัญ เป็นดังนี้

Element selector
Class selector , Pseudo-Classes
ID selectors
Inline

Element selector (ค่าความสำคัญน้อยที่สุด) Class selector , Pseudo-classes ID selectors Inline style(ค่าความสำคัญมากที่สุด)
แบ่งคะแนนเค่าความสำคัญเป็นแบบนี้ 0,0,0,0 (เลข 4 ตัว Style 4 กลุ่ม )
(อธิบายเพิ่มเติมในตัวอย่าง)

Example

HTML file

กำหนด Style แบบ Inline style ให้ตัวอักษรมีสีแดง

point : 1,0,0,0

CSS file

กำหนด Style แบบ Element selector ให้ตัวอักษรมีสีเหลือง

point : 0,0,0,1

Output

Inline style มีค่าความสำคัญมากสุด ผลลัพธ์ออกมา ตัวอักษรจะมีสีแดง

HTML file

กำหนด Style แบบ Inline style ให้ตัวอักษรมีสีแดง

point : 1,0,0,0

CSS file

กำหนด Style แบบ Element selector ให้ตัวอักษรมีสีเหลือง

point : 0,0,0,1

Output

Inline style มีค่าความสำคัญมากสุด ผลลัพธ์ออกมา ตัวอักษรจะมีสีแดง
