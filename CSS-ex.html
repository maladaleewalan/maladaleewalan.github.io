HTML file

กำหนด Style แบบ Inline style ให้ตัวอักษรมีสีแดง

1,0,0,0

CSS file

กำหนด Style แบบ Element selector ให้ตัวอักษรมีสีเหลือง

0,0,0,1

Output

Inline style มีค่าความสำคัญมากสุด ผลลัพธ์ออกมา ตัวอักษรจะมีสีแดง

HTML file

div มี id = test

CSS file

กำหนด style

div#test h1 : 0,1,0,2
div h1 : 0,0,0,2
h1 : 0,0,0,1

Output

โดยจะเห็นว่า div#test h1 มีการกำหนดแบบเจาะจงที่สุด และมีค่าความสำคัญมากที่สุด ตัวอักษรจึงออกมาเป็นสีส้ม

CSS file

การกำหนด style โดย html , body , h1 ต่างก็เป็น Element selector ทั้ง 2 selector นี้จึงมีค่าความสำคัญเท่ากัน

html h1 : 0,0,0,2
body h1 : 0,0,0,2

Output

ทั้ง 2 selector มีค่าความสำคัญเท่ากัน จึงเลือก selector ที่อยู่ด้านล่าง ตัวอักษรจึงออกมาเป็นสีม่วง
